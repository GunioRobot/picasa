= Picasa

Simple google picasa managment.
Only for public albums so far.

= Installation

  gem install rafaels-picasa

== Usage

  user = Picasa::User.new('google_username')

  albums = user.albums # returns user's public albums

  user.photos(:album_id => 1234567) # returns all photos from album with given id

  or you can call photos directly from an album

  albums.first.photos # returns all photos from first album
